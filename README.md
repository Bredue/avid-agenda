# AVID Agenda

Check out the project [here](https://bredue.github.io/avid-agenda/)

A fast, lightweight web app that makes creating, storing, and displaying classroom agendas simple.

Designed with AVID classrooms in mind, the app stores all agenda data locally in your web browser, so no account, sign-in, or authentication is required. Everything stays on your device, allowing teachers to quickly create and access agendas whenever they need them. As long as you're using the same device and don't clear your browser data, your agendas will remain available.

The project is inspired by the AVID (Advancement Via Individual Determination) framework, which focuses on preparing K–12 students for college, careers, and life through organization, structure, and student readiness. Learn more about AVID at https://www.avid.org.

<br>

> **Note:** If you're using Firefox, ensure that strict privacy settings are disabled so the app can access local storage. The app cannot be used in private/incognito browsing mode. Because all data is stored locally, clearing your browser data, reinstalling your browser, or switching to a different device will permanently remove your saved agendas.
